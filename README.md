# StudyPro

[![Status](https://img.shields.io/badge/status-active-brightgreen)](#) [![License](https://img.shields.io/badge/license-MIT-blue)](#)

StudyPro est une application web front-end simple (HTML / CSS / JavaScript) 
destinée à aider les étudiants à organiser leurs cours, tâches et révisions. 
Le projet est une application statique composée des fichiers principaux : `index.html`, `style.css` et `script.js`.
